Chapter 5 (Equality): Function Summary

1 Overview

This document summarizes the functions defined in HoTTRijke/chapter5_eq.lean, in namespace chapter5_myeq.

The file introduces a custom identity type MyEq, proves basic equality properties, defines path composition operations, and develops transport and higher coherence constructions.

2 Core Equality Type

2.1 MyEq

Name Signature Description
MyEq.refl {α : Type} → (a : α) → MyEq a a Constructor witnessing reflexivity of equality.

3 Function Summary

3.1 Equality Induction and Basic Laws

Function Signature Description
ind_eq {α : Type} → {a : α} → (P : (x : α) → (a ≡ x) → Type) → P a (MyEq.refl a) → ((x : α) → (p : a ≡ x) → P x p) Eliminator/induction principle for MyEq, reducing all equality proofs to reflexivity.
myEq_symm {α : Type} → {a x : α} → (a ≡ x) → (x ≡ a) Symmetry of equality (path inversion).
myEq_symm_refl {α : Type} → (a : α) → myEq_symm (MyEq.refl a) ≡ (MyEq.refl a) Shows symmetry preserves reflexivity judgmentally/propositionally.
myEq_refl {α : Type} → {a : α} → (a ≡ a) Reflexivity of equality.
myEq_trans {α : Type} → {a x y : α} → (a ≡ x) → (x ≡ y) → (a ≡ y) Transitivity of equality.
concat_eq {α : Type} → {x y z : α} → (x ≡ y) → (y ≡ z) → (x ≡ z) Equality concatenation (path composition), equivalent to transitivity.
concat_assoc {α : Type} → {a b c d : α} → (p : a ≡ b) → (q : b ≡ c) → (r : c ≡ d) → (p • (q • r)) ≡ ((p • q) • r) Associativity of path concatenation.
concat_right_unit {α : Type} → {a b : α} → (p : a ≡ b) → (p • (MyEq.refl b)) ≡ p Right identity law for concatenation.
concat_left_unit {α : Type} → {a b : α} → (p : a ≡ b) → ((MyEq.refl a) • p) ≡ p Left identity law for concatenation.
left_unit {α : Type} → {a b : α} → (p : a ≡ b) → (a ≡ b) Repackages left-unit composition as a term of the same equality type.
right_unit {α : Type} → {a b : α} → (p : a ≡ b) → (a ≡ b) Repackages right-unit composition as a term of the same equality type.

3.2 Action on Paths and Composition Properties

Function Signature Description
ap {α : Type} → {β : Type} → (f : α → β) → (x y : α) → (p : x ≡ y) → (f x ≡ f y) Applies a function to both sides of an equality (functorial action on paths).
sym_contat_distributive {α : Type} → {x y z : α} → (p1 : x ≡ y) → (p2 : y ≡ z) → myEq_symm (p1 • p2) ≡ (myEq_symm p2) • (myEq_symm p1) Inversion distributes over concatenation in reverse order.
inv_con {α : Type} → {a b c : α} → (p : a ≡ b) → (q : b ≡ c) → (r : a ≡ c) → ((p • q) ≡ r) → (q ≡ (myEq_symm p • r)) Solves for q from a composed path equation by composing with inverse of p.
con_inv {α : Type} → {a b c : α} → (p : a ≡ b) → (q : b ≡ c) → (r : a ≡ c) → ((p • q) ≡ r) → (p ≡ r • (myEq_symm q)) Solves for p from a composed path equation by composing with inverse of q.
ap_id {α : Type} → (x y : α) → (p : x ≡ y) → p ≡ (ap (fun (x : α) => x) x y p) Shows ap on identity function returns the same path.
ap_comp {α : Type} → {β : Type} → {γ : Type} → (f : α → β) → (g : β → γ) → (x y : α) → (p : x ≡ y) → ap g (f x) (f y) (ap f x y p) ≡ ap (g ∘ f) x y p Functoriality of ap with respect to composition.

3.3 Sigma Paths and Transport

Function Signature Description
unique_ref {α : Type} → (x y : α) → (p : x ≡ y) → (⟨x, MyEq.refl x⟩ : Σ (y' : α), x ≡ y') ≡ ⟨y, p⟩ Uniqueness-style result for equality witnesses packaged in a sigma type.
transport {α : Type} → {x y : α} → (β : (x' : α) → Type) → (x ≡ y) → (β x → β y) Transports terms along equalities in dependent types.
lift_β (α : Type) → (β : (a : α) → Type) → (a x : α) → (b : β a) → (p : a ≡ x) → (⟨a, b⟩ : Σ a, β a) ≡ ⟨x, transport β p b⟩ Lifts base equality to an equality in a dependent pair (Sigma) via transport.

3.4 Higher Coherence

Function Signature Description
maclane_pentagon {α : Type} → (a b c d e : α) → (p : a ≡ b) → (q : b ≡ c) → (r : c ≡ d) → (s : d ≡ e) → Type Encodes the Mac Lane pentagon coherence construction for associativity paths. The current implementation builds intermediate path terms and targets the pentagon witness type.
